Crayfish

PDF templating for rails, allowing you to not only layout PDF's with familiar HTML tags, but also integrate it with rails HTML view templating and view testing. This automatic page generator is the easiest way to create beautiful pages for all of your projects. Author your page content here using GitHub Flavored Markdown, select a template crafted by a designer, and publish. After your page is generated, you can check out the new branch:

Install

$ gem install crayfish

Prawn havent been gemified for a while, so you need our latest snapshot. Add this to your Gemfile:

gem 'prawn', :git => 'git://github.com/teknobingo/prawn.git', :branch => 'master'
